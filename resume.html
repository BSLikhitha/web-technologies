Financial Transaction Security using AES and Machine Learning Approach.

This project presents a dual-layer security framework for protecting digital financial transactions from data breaches and fraud. It uses AES encryption to ensure data confidentiality and a Decision Tree–based machine learning model for real-time fraud detection. Transaction files are encrypted before storage and securely decrypted for analysis in a protected environment. The system evaluates features like amount, time, and location to classify transactions as legitimate or fraudulent. Suspicious transactions are blocked and verified via OTP, while valid transactions are processed and re-encrypted for secure storage.
